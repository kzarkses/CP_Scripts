## Overall verdict

Architecture is sound; the perf discipline (idle throttle, measure cache, format caches,
zero-alloc rules) is ahead of the REAPER-toolkit sector. But several roadmap items marked
`[x]` are bypassed or buggy, real correctness bugs exist in widgets, and "true idle"
does not exist yet. Most fixes are local.

## Sector techniques worth keeping in mind

- **egui**: repaint-on-demand + `request_repaint_after` deadlines → adopted (P2).
- **Scythe/Lokasenna**: per-z-layer offscreen buffers + composite blit — the simpler,
  REAPER-proven alternative to roadmap 3.1 dirty rects. Recommended for CP Studio
  (meters layer isolated from static panels).
- **rtk**: attribute invalidation levels (hover must never invalidate geometry);
  `gfx.setfont` is "extremely expensive" → slot guards adopted (P15).
- **microui**: LRU pools for retained state → adopted (P11); hover-root with one-frame
  lag for popup occlusion — worth adding if CP Studio stacks popups.
- **ImGui**: ListClipper (F1), SkipItems early-out, zero-malloc idle frames.
